لغة HTML

ما هي HTML؟

HTML (لغة ترميز النص التشعبي) هي العمود الفقري لأي موقع ويب. تستخدم لإنشاء هيكل صفحات الويب وتحديد العناصر المختلفة مثل العناوين والفقرات والصور والروابط.

الأساسيات:

  • HTML تعني HyperText Markup Language
  • ليست لغة برمجة، بل لغة ترميز
  • تتكون من عناصر (Elements) تحيط بالمحتوى
  • كل عنصر له وسم فتح ووسم إغلاق

مثال على كود HTML:

<!DOCTYPE html>
<html>
<head>
    <title>عنوان الصفحة</title>
</head>
<body>
    <h1>مرحباً بالعالم!</h1>
    <p>هذه هي أول صفحة ويب لي.</p>
</body>
</html>

لغة CSS

ما هي CSS؟

CSS (أوراق الأنماط المتتالية) هي لغة تستخدم لوصف مظهر وترتيب صفحات الويب. تتحكم في الألوان والخطوط والمسافات والتصميم العام.

الأساسيات:

  • CSS تعني Cascading Style Sheets
  • تستخدم لتنسيق وتجميل صفحات HTML
  • تعمل على مبدأ الانتقالية والتوريث
  • يمكن كتابتها داخلياً أو خارجياً

مثال على كود CSS:

body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    color: #333;
}

h1 {
    color: #0066cc;
    text-align: center;
    margin-bottom: 20px;
}

.container {
    width: 80%;
    margin: 0 auto;
    padding: 20px;
}

لغة JavaScript

ما هي JavaScript؟

JavaScript هي لغة برمجة تستخدم لجعل صفحات الويب تفاعلية. تمكنك من إنشاء محتوى ديناميكي، تحكم في الوسائط المتحركة، وتتفاعل مع المستخدم.

الأساسيات:

  • لغة برمجة عالية المستوى ومفسرة
  • تستخدم على جانب العميل (Client-side)
  • تدعم البرمجة كائنية التوجه (OOP)
  • يمكن استخدامها أيضاً على جانب الخادم (Node.js)

مثال على كود JavaScript:

// تغيير نص العنصر عند النقر عليه
document.getElementById("myButton").onclick = function() {
    document.getElementById("demo").innerHTML = "تم النقر!";
};

// دالة بسيطة لجمع عددين
function addNumbers(a, b) {
    return a + b;
}

// استدعاء الدالة
let result = addNumbers(5, 3);
console.log("الناتج: " + result);

مقارنة بين Java و JavaScript

لغة Java

  • النوع: لغة برمجة كائنية التوجه
  • الاستخدام: تطبيقات سطح المكتب، تطبيقات الويب الخلفية، تطبيقات Android
  • التنفيذ: تحتاج إلى JVM (Java Virtual Machine)
  • النمط: لغة مكتوبة بقوة (Strongly Typed)
  • البناء: تحتاج إلى تجميع (Compilation)
  • الأداء: أسرع بشكل عام

الفرق

على الرغم من تشابه الاسمين، إلا أن Java و JavaScript لغتان مختلفتان تماماً:

  • Java لغة برمجة مستقلة، JavaScript لغة برمجة نصية للويب
  • Java تحتاج إلى تجميع، JavaScript مفسرة مباشرة في المتصفح
  • Java تستخدم لبناء تطبيقات كاملة، JavaScript تستخدم لجعل صفحات الويب تفاعلية
  • Java لها أنواع بيانات صارمة، JavaScript لها أنواع بيانات ديناميكية

لغة JavaScript

  • النوع: لغة برمجة نصية (Scripting)
  • الاستخدام: صفحات الويب التفاعلية، تطبيقات الويب، تطبيقات الهاتف (React Native)
  • التنفيذ: تعمل مباشرة في المتصفح
  • النمط: لغة مكتوبة بشكل ديناميكي (Dynamically Typed)
  • البناء: لا تحتاج إلى تجميع
  • الأداء: أسرع في التطوير
